Meogeoyori API (먹어요리)

Supabase SDK를 통해 호출되는 가상의 RESTful API 명세서입니다.

Shorts (숏폼 영상)
GET
/shorts
피드 영상 목록 조회
Parameters
NameDescription
page
integer
(query)
 페이지 번호 (기본값 1)
limit
integer
(query)
 한 번에 가져올 영상 수 (기본값 10)
Responses

200 - OK

{
  "data": [
    {
      "id": "uuid-string",
      "video_url": "https://...",
      "title": "초간단 계란 볶음밥",
      "like_count": 150,
      "user": {
        "username": "요리왕초보"
      }
    }
  ]
}
POST
/shorts/{videoId}/like
좋아요 토글 (추가/취소)
Parameters
NameDescription
videoId * required
string(uuid)
(path)
 대상이 되는 영상의 ID
Responses

200 - OK

{
  "message": "좋아요 상태 변경됨",
  "is_liked": true
}
POST
/shorts/{videoId}/bookmark
레시피 북마크 토글
Parameters
NameDescription
videoId * required
string(uuid)
(path)
 대상이 되는 영상의 ID
Responses

200 - OK

{
  "message": "북마크에 추가되었습니다",
  "is_bookmarked": true
}
